# hexo-theme-zen

A minimalist, clean Hexo theme for personal blogs.

## Features

- Clean, minimal design
- Dark/light mode toggle
- Tag support
- Twikoo & Beaudar comment systems
- Responsive layout

## Install

```bash
git clone https://github.com/MichaelSatoshi1983/hexo-theme-zen themes/zen
```

## Configure

Edit `themes/zen/_config.yml`.
